Shut The Box!

Welcome to a web-based dice roller for the game "Shut the box!" This application does not track which numbers are closed, this is only a replacement for rolling the dice. To play, just press the "Roll Dice" button, and two d6 dice will be "rolled." Below the generated values, you will see which numbers can be closed. If you can't close at least one number per roll, you lose... If you manage to close all of the numbers, you win! Good luck!
